
# Convo — AI Front Desk for Small Shops (Demo)

This repository is part of **Convo**, an AI-powered front desk solution built for salons and local service businesses.

⚠️ **Important Note**
The website https://www.convoaiservices.com/ is only a demo and marketing showcase of the product.
It is not the full production application or complete codebase.

## What Convo Does

Convo is designed to automate salon front-desk operations by:

* Handling appointment bookings via AI chat, voice, and messaging
* Acting as a virtual receptionist for customer calls and inquiries
* Managing services, staff, availability, and scheduling logic
* Reducing missed calls and manual booking overhead for shop owners

## About the Demo Website

The public website demonstrates:
* How customers interact with the AI to book appointments
* The overall user experience and product flow
* Example conversations and booking behavior

The demo **does not include**:
* Full backend logic
* Production databases
* Internal AI workflows
* Business-specific integrations

Purpose of This Repository
This repository contains (or is intended to contain):
* Core backend and API logic
* AI conversation handling
* Scheduling and booking systems
* Internal tools and integrations

Use the demo site only as a **reference for product behavior and UX**, not as a representation of the full system.

## Status
